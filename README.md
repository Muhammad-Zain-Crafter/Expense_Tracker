# Expense Tracker Backend

A **backend application** for managing personal finances, built with **Node.js**, **Express.js**, and **MongoDB**.  
This project allows users to track income, expenses, set budgets, and get monthly summaries of their financial activity.

---

## Features

- **User Management**
  - Register and login users
  - JWT authentication
  - Role-based access (User, Admin - planned)
  
- **Expense Management**
  - Add, update, and delete expenses
  - Categorize expenses for easy tracking

- **Income Management**
  - Add, update, and delete income sources
  - Track multiple income streams

- **Budget Management**
  - Set monthly budgets
  - Limit spending per category
  - Notifications if limits exceed (planned)

- **Monthly Summary**
  - Overview of total income, expenses, and savings
  - Categorized breakdown of expenses
  - Visual summaries ready for frontend integration

---

## Tech Stack

- **Backend:** Node.js, Express.js  
- **Database:** MongoDB, Mongoose  
- **Authentication:** JWT (JSON Web Tokens)  
- **Environment Variables:** dotenv
